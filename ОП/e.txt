importFromFile. Функция запрашивает путь к файлу, из которого требуется импортировать данные. Затем она открывает этот файл и считывает из него данные, заполняя ими массив структур Student. После завершения чтения файла количество записей в БД обновляется.
exportToFile. Функция запрашивает путь к файлу, в который требуется экспортировать данные. Затем она открывает этот файл и записывает в него данные из массива структур Student.
displayStudentList. Функция выводит на экран все записи из массива структур Student.
printStudentList. Функция выводит на принтер все записи из массива структур Student.
displaySortedStudentList. Функция сначала сортирует массив структур Student, а затем выводит его на экран.
sortStudentList. Функция сортирует массив структур Student в зависимости от выбранного пользователем режима сортировки.
specifyFilePath. Функция запрашивает у пользователя путь к файлу и сохраняет его.
getInput. Функция запрашивает у пользователя ввод и возвращает его.
swapStudents. Функция меняет местами две структуры Student в массиве.
sortByName, sortByPatronymic, sortByBirthYear, sortByCourse, sortByMark. Эти функции сортируют массив структур Student по соответствующим полям.
changeStudent. Функция запрашивает у пользователя номер студента, данные которого требуется изменить, а затем запрашивает новые данные и обновляет соответствующую структуру Student.
deleteStudent. Функция запрашивает у пользователя номер студента, которого требуется удалить, а затем удаляет соответствующую структуру Student из массива и обновляет количество записей в БД.
importStudentList. Функция импортирует список студентов из файла.
exportStudentList. Функция экспортирует список студентов в файл.
isStudents. Функция проверяет, есть ли студенты в списке.
transformStudentList. Функция преобразует список студентов.
clearInput. Функция очищает ввод пользователя.
clearDisplay. Функция очищает консоль.
printMenu. Функция отображает меню программы.
